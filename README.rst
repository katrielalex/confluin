A confluence checker for Tamarin equational theories

Usage
-----

    $ confluin foo.spthy

and see what it says!


Installation
------------

To install use pip:

    $ pip install git+https://github.com/katrielalex/confluin

Or clone the repo:

    $ git clone https://github.com/katrielalex/confluin.git
    $ python setup.py install
